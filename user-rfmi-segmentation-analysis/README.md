# 基于 RFMI 模型的用户精细化分层与营销 ROI 分析

## 项目背景
对电商平台用户进行 RFM + 意向得分(I) + 摩擦值(Friction) + 收入分层，实现精准用户分群，提升营销 ROI。

## 技术工具
- MySQL：特征工程、用户分层、ROI测算
- PowerBI：可视化看板、EDA探索分析
- Excel：数据预处理

## 核心指标
- R：最近登录
- F：购买频次
- M：消费金额
- I：用户意向评分
- Friction：购买摩擦值

## 分层结果
- 核心VIP、重要价值用户、高潜流失客、纠结土豪、隐形活跃者、羊毛党等
- 优化后 ROI 从 4.04% 提升至 29.89%

## 项目文件
- /sql：完整MySQL建模脚本
- /powerbi：可视化看板
- /data：数据集
- /image：图表截图

## 核心业务洞察
1. 高收入用户≠高消费，消费与收入无强相关
2. 邮件营销对城市用户无效
3. 高净值用户存在极高购买摩擦
4. RFMI 模型比传统 RFM 精准度大幅提升
