# 📊 基于 RFMI 模型与多维交叉分析的用户精细化分层与 ROI 提升方案

> **分析工具**：MySQL、Power BI
> **核心产出**：通过引入意向分(I)与摩擦力(Friction)，优化传统 RFM 模型，精准圈选人群，使**营销 ROI 从 4.04% 爆炸性提升至 29.89%**。

## 💡 核心商业洞察 (Core Business Insights)
1. **向上纠偏**：发掘被低估的“纠结土豪”与“隐形活跃者”，打破“收入决定消费力”的固有认知。
2. **向下预警**：提前揪出原安全区（重要保持用户）中的21名“高潜流失大客”，紧急止损。
3. **精准止损**：精准剔除“羊毛党”，避免无效营销资源浪费。
4. **体验阻力排查**：发现最高净值人群（纠结土豪）正面临严重的购物体验阻力（Friction飙升至28.1），急需产品端排查。

## 🛠️ 分析框架与特征工程 (Methodology)
本项目采用 **Min-Max 归一化**方法，将多维指标映射为 0-100 分。
* **I_Score (用户意向评分)**：耗时归一化 * 0.5 + 浏览页数归一化 * 0.5
* **Friction (购买摩擦值)**：浏览页数 / (购买频次 + 1)
* **RFM_Score (加权综合得分)**：0.2*R + 0.3*F + 0.5*M
* **Final_Score**：引入意向权重修正 `RFM_Score * (1 + I_Score / 500)`
